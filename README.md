# 🎓 Student Result Manager App

A simple Python-based Student Result Management System that allows users to manage student records, calculate grades, check results, update marks, and identify the class topper.

## 🚀 Features

* ➕ Add Student
* ❌ Delete Student
* 👀 View All Students
* 🏆 Calculate Student Grade
* ✏️ Update Student Marks
* 📊 Calculate Student Percentage
* ✅ Check Pass/Fail Result
* 🥇 Display Class Topper
* 🚪 Exit Application

## 🛠️ Technologies Used

* Python 3
* Dictionary Data Structure
* Loops
* Conditional Statements
* User Input Handling

## 📋 Menu Options

| Option | Description              |
| ------ | ------------------------ |
| 1      | Add Student              |
| 2      | Delete Student           |
| 3      | View Student Records     |
| 4      | Show Student Grade       |
| 5      | Update Marks             |
| 6      | Calculate Percentage     |
| 7      | Check Result (Pass/Fail) |
| 8      | Show Topper              |
| 9      | Exit Program             |

## 🎯 Grade Criteria

| Marks    | Grade |
| -------- | ----- |
| 95 - 100 | A+    |
| 75 - 94  | A     |
| 50 - 74  | B     |
| 25 - 49  | C     |
| Below 25 | Fail  |

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Ankit0830/Student_Result_Manager_App.git
```

2. Navigate to the project folder:

```bash
cd Student_Result_Manager_App
```

3. Run the Python file:

```bash
python student_result_manager.py
```
